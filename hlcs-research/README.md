# Hybrid Post-Quantum Commitment Schemes for Latency-Constrained Financial Systems

This repository contains a Rust simulation and analysis framework that empirically implements, benchmarks, and statistically validates the HLCS protocol against its hash-only and lattice-only baselines.

## Prerequisites
- Rust stable toolchain ≥ 1.78
- `make`

## Usage
To build and run all experiments end-to-end and generate all figures and tables:
```sh
make all
```

To run unit tests:
```sh
make test
```

To clean the generated data and artifacts:
```sh
make clean
```

All randomness is deterministically seeded from `experiments/config.toml` for complete reproducibility.

## Outputs
- `figures/`: 18 figures (.png) generated by the framework. Note that F04 and F05 separately address and fix the original paper's captioning mix-up regarding the latency heatmap and EUR/USD tick data. F04 also properly implements the mislabeled placeholder figure.
- `tables/`: 16 tables (.csv, .tex, .md) with data generated by the framework.
- `report/results_report.md`: Combined Markdown report summarizing the findings.
